# Claude Code 101 — Workshop Setup Guide

Pre-workshop setup instructions for the Claude Code 101 AI Champions workshop.

Open `index.html` in a browser, or visit the hosted version for step-by-step setup instructions with screenshots.

## Steps covered
1. Install Claude Code Desktop App
2. Create your Claude folder
3. Initialize Git
4. Create your CLAUDE.md file
5. Enable Co-work & restart
6. Connect MCPs (Google Drive, Gmail, Calendar, Slack)
